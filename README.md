# unzip-tool

ZIP 파일을 브라우저에서 바로 푸는 도구. 드래그 앤 드롭, 한글 파일명 자동 복원, 선택 해제, 미리보기 지원.

🔗 **라이브 데모**: https://commme.github.io/unzip-tool/

## 결과물

- **`index.html`** — 단일 HTML. ZIP 목록을 즉시 표시하고 원하는 파일만 골라서 해제.

## 핵심 기능

- **지연 해제 (Lazy Extraction)** — ZIP 중앙 디렉토리(Central Directory)만 먼저 파싱해서 목록을 즉시 표시. 파일 하나당 다운로드 버튼을 눌러야 실제 압축 해제
- **한글 파일명 복원** — UTF-8 플래그 확인 + CP949(EUC-KR) 자동 폴백
- **파일명 sanitize** — TAB·제어문자·`:*?"<>|` 등 OS 금지 문자 `_`로 치환 (다운로드 실패 방지)
- **선택 해제 + 검색** — 체크박스로 원하는 것만, 파일명 검색으로 필터링
- **묶음 다운로드** — 선택한 파일을 새 ZIP으로 재압축해 1회 다운로드
- **미리보기** — 텍스트(UTF-8/EUC-KR 자동 감지) / 이미지 인라인 표시
- **Zip64 지원** — 4GB 이상 아카이브도 처리 가능

## 사용법

1. https://commme.github.io/unzip-tool/ 접속
2. ZIP 파일을 드롭 존에 드래그 또는 클릭해서 선택
3. 파일 목록이 뜨면 원하는 파일 체크 → **선택 파일 일괄 다운로드 (ZIP)** 또는 **개별 다운로드**
4. 필요 시 **미리보기**로 내용 확인 후 저장

## 사용된 도구

| 영역 | 라이브러리/도구 |
|------|-----------------|
| ZIP 해제 (deflate) | [fflate](https://github.com/101arrowz/fflate) 0.8.2 |
| 중앙 디렉토리 파싱 | 직접 구현 (Zip64 + CP949 폴백) |
| UI 폰트 | Pretendard |

## 보안

- 모든 처리는 **브라우저 로컬에서만** — 파일이 외부로 전송되지 않음
- CSP 메타 태그로 외부 스크립트 화이트리스트 (`cdn.jsdelivr.net`만 허용)
- 사용자 입력은 `textContent` 처리 (XSS 방지)
- 로그인/계정/쿠키 전혀 없음
- 암호화된 ZIP(AES / ZipCrypto)은 **지원하지 않음** — 명시적 에러 표시

## 한계

- **AES 암호화 ZIP 미지원** — fflate가 암호화를 지원하지 않음. 필요하면 7-Zip 등 데스크톱 도구 사용
- **권장 크기**: 2GB 이하 — 브라우저 메모리 한계. 이보다 크면 모바일/구형 PC는 실패 가능
- **Compression method**: `stored(0)` + `deflate(8)`만 지원 (ZIP 99% 이상이 이 두 가지)
